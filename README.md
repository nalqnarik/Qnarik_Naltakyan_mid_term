# Hotel Booking Demand — EDA & Feature Engineering

Այս պրոյեկտը իրականացվել է որպես 2-րդ միջանկյալ քննության աշխատանք։
Նպատակը տվյալների հետազոտական վերլուծության (EDA) և
Feature Engineering-ի իրականացումն է իրական տվյալների վրա։

## Տվյալների աղբյուր
Տվյալները վերցված են Kaggle հարթակից՝  
Hotel Booking Demand dataset։

Տվյալները պարունակում են ինչպես քանակական, այնպես էլ որակական
հատկանիշներ, ինչպես նաև բաց թողնված արժեքներ։

## Օգտագործված գործիքներ
- Python
- Pandas
- Matplotlib
- Google Colab
- GitHub

## Կատարված աշխատանքներ

### 1. Exploratory Data Analysis (EDA)
- Տվյալների կառուցվածքի ուսումնասիրություն
- Բաց թողնված արժեքների վերլուծություն
- Քանակական հատկանիշների բաշխումների ուսումնասիրություն
  (histogram, boxplot)
- Քանակական հատկանիշների փոխկապակցության ուսումնասիրություն
  (scatterplot)
- Որակական հատկանիշների բաշխումների վիզուալիզացիա
- Յուրաքանչյուր քայլի համար գրվել են մեկնաբանություններ և հետևություններ

### 2. Feature Engineering
- Ստեղծվել են նոր հատկանիշներ՝
  - մնալու ընդհանուր տևողություն (total_nights)
  - հյուրերի ընդհանուր քանակ (total_guests)
- Կիրառվել է one-hot encoding որակական սյուների համար
- Նոր հատկանիշների համար իրականացվել են վիզուալիզացիաներ
- Կատարվել են մեկնաբանություններ և հետևություններ

## Եզրակացություն
Տվյալների հետազոտական վերլուծության արդյունքում պարզվել է,
որ տվյալների հավաքածուն ունի բարդ կառուցվածք, արտածայր արժեքներ
և անհամաչափ բաշխումներ։

Feature Engineering-ի միջոցով ստեղծված նոր հատկանիշները
ավելացնում են տվյալների տեղեկատվական արժեքը և կարող են
օգտագործվել հետագա վերլուծության կամ մոդելավորման փուլերում։
